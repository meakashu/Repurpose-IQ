## üöÄ API Endpoints Summary

### Monitoring (`/api/monitoring/*`)
- `GET /status` - Get monitoring status
- `POST /start` - Start monitoring
- `POST /stop` - Stop monitoring
- `POST /add-molecule` - Add molecule to track
- `POST /remove-molecule` - Remove molecule
- `GET /alerts` - Get alerts
- `POST /alerts/:id/read` - Mark as read

### Vision (`/api/vision/*`)
- `POST /analyze-image` - Analyze image
- `POST /recognize-structure` - Recognize molecular structure
- `POST /process-document` - Process document

### Workflows (`/api/workflows/*`)
- `POST /` - Create workflow
- `GET /` - List workflows
- `GET /:id` - Get workflow
- `POST /:id/execute` - Execute workflow
- `PUT /:id` - Update workflow
- `DELETE /:id` - Delete workflow

### Suggestions (`/api/suggestions/*`)
- `GET /` - Get suggestions
- `GET /popular` - Get popular queries
- `POST /save` - Save query

### Sentiment (`/api/sentiment/*`)
- `POST /analyze` - Analyze sentiment
- `GET /history/:molecule` - Get sentiment history

### Predictions (`/api/predictions/*`) - Python
- `POST /repurposing-success` - Predict success
- `POST /market-forecast` - Forecast market
- `POST /patent-expiry-impact` - Patent impact

### Graph (`/api/graph/*`) - Python
- `POST /repurposing-paths` - Find paths
- `GET /drug-network/:id` - Get network
- `GET /similar-drugs/:id` - Find similar drugs
- `POST /add-node` - Add node
- `POST /add-relationship` - Add relationship
